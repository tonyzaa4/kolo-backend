# kolo-backend

Серверна частина мобільного застосунку **«Kolo»** — зручного фінансового трекера для управління платними підписками, аналітики витрат та отримання автоматичних нагадувань.

## 🛠 Стек технологій
* **Мова:** Python 3.10+
* **Фреймворк:** FastAPI
* **База даних:** MySQL
* **ORM та Міграції:** SQLAlchemy + Alembic
* **Тестування:** Pytest

---

## 🚀 Інструкція із запуску (для розробників)

Ця інструкція допоможе вам розгорнути локальну копію сервера на вашому комп'ютері.

### Крок 1: Клонування репозиторію
Відкрийте термінал і виконайте команду:
```bash
git clone [https://github.com/ВАШ_НІК/kolo-backend.git](https://github.com/ВАШ_НІК/kolo-backend.git)
cd kolo-backend
```

Крок 2: Створення та активація віртуального середовища 
Це необхідно для ізоляції бібліотек проєкту від вашої системи.

```bash
python -m venv venv
```
**Активація:**
* Для Windows: `venv\Scripts\activate`
* Для macOS / Linux: `source venv/bin/activate`

*(Після активації в терміналі має з'явитися префікс `(venv)`).*
`

Крок 3: Встановлення залежностей
Встановіть усі необхідні пакети з файлу конфігурації:
```
pip install -r requirements.txt
```

Крок 4: Запуск локального сервера
Для запуску сервера в режимі розробника (з автоматичним перезавантаженням при зміні коду) виконайте:
```
uvicorn main:app --reload
```
API Документація
FastAPI автоматично генерує інтерактивну документацію (Swagger UI) на основі нашого коду та API-контракту.
Після успішного запуску сервера перейдіть у браузері за посиланням:
```
 https://www.google.com/search?q=http://127.0.0.1:8000/docs
```
Тут ви зможете переглянути всі доступні маршрути (endpoints) та відправляти тестові запити без використання сторонніх програм (Postman тощо).
